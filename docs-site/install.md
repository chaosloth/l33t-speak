# Installation

## From VS Code Marketplace

1. Open VS Code
2. Press `Cmd+Shift+X` (macOS) or `Ctrl+Shift+X` to open Extensions
3. Search for **"L33t Speak"**
4. Click **Install**

Or install from the command line:

```bash
code --install-extension chaosloth.l33t-speak
```

## From VSIX File

Download the latest `.vsix` from [GitHub Releases](https://github.com/chaosloth/l33t-speak/releases), then:

```bash
code --install-extension l33t-speak-0.1.0.vsix
```

## Requirements

- **macOS 10.15+** or **Windows 10/11**
- VS Code 1.74 or later

## Permissions (macOS)

On first use, macOS will prompt you to grant permissions to VS Code:

1. **Microphone** — System Settings > Privacy & Security > Microphone > VS Code ✓
2. **Speech Recognition** — System Settings > Privacy & Security > Speech Recognition > VS Code ✓

If you accidentally denied access, toggle it off and on in System Settings.
